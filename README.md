## 📚 What We Are Learning (Course Modules)

This course is divided into three main parts:

### 1. Machine Learning (ML)
Learning how to make computers learn from data without hard-coding rules.
* Data Cleaning and Preprocessing
* Regression Models (Predicting numbers like prices)
* Classification Models (Predicting categories like Yes/No)
* Scikit-Learn, XGBoost, and Decision Trees

### 2. Deep Learning (DL)
Learning how to build Artificial Neural Networks that work like the human brain.
* Basic Neural Networks (ANN)
* Computer Vision and Image Processing (CNN)
* Text and Natural Language Processing (RNN / LSTM)
* TensorFlow and Keras

### 3. Applied Artificial Intelligence (AI)
Using ML and DL to solve real-world industry problems.
* Generative AI basics
* Building end-to-end AI pipelines
* Model deployment and testing
